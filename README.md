
# Overtime Improvement Proposals (OIPs)


Overtime Improvement Proposals (OIPs) describe standards, processes and provide general guidelines or information to the Overtime Community.

## Create a OIP

Follow this [step-by-step tutorial](https://medium.com/@Thales.Academy/how-to-use-github-to-submit-a-tip-e7d0047b54fd) to learn about the process of creating a OIP( formerly TIP).
